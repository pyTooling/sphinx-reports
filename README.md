# Sphinx Reports

## Supported Report Formats

* Python documentation coverage (doc-string coverage)
  * `docstr_coverage`
  * `interrogate`
* Python code coverage (line/statement and branch coverage)
  * `coverage.py`
* Unit test summary
  * `pytest`
* Static type analysis
  * `mypy`
